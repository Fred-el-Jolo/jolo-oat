    What is oat?

    oat is an ultra-lightweight (~8KB combined), zero-dependency UI component library using semantic HTML, plain CSS, and native Web APIs.

    Does it require a build step?

    No build step required for usage. Simply include oat.min.css and oat.min.js in your HTML. The library uses standard Web Components and the Popover API.

    What browsers are supported?

    oat targets modern evergreen browsers that support Web Components, Popover API, and light-dark() CSS function. Safari 17.4+, Chrome 114+, Firefox 117+.

    Is it framework agnostic?

    Yes. Because oat is built on standard web platform features, it works with any framework or no framework at all.

    Dropdown

    The <ot-dropdown> WebComponent wraps a trigger button and a popover menu. It handles dynamic viewport-aware positioning, keyboard navigation (↑↓ Home End), and ARIA state.

    Tooltip

    Pure CSS tooltips via data-tooltip="text". A JS enhancement automatically promotes title attributes to data-tooltip. Hover delay is 700ms.

    Hover over me

    Toast

    Imperative API via ot.toast(message, title, opts). Options: variant (info, success, danger, warning), placement (top/bottom × left/center/right), duration ms. Pauses on hover.
